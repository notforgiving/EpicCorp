FAQ

  • Как сменить город для осуществления покупок?

    При первом запуске сайта отображается всплывающее окно приветствия, в котором можно выбрать свой город. Также регион можно сменить, щелкнув по установленному на данный момент городу в верхнем левом углу страницы сайта. Город можно выбрать из предложенного списка или используя поле «Введите свой город».
